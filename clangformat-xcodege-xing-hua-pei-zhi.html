    ClangFormat-Xcode个性化配置
    1. 下载ClangFormat-Xcode
    2. 解压，进入文件夹，打开项目，command+b安装插件，重启xcode
    3. 个性化配置，打开终端运行
    ./bin/clang-format -style=llvm -dump-config > xx
    4. 编辑xx,参照clang documentation进行配置
    5. 复制xx到用户根目录，终端运行
    cp xx ~/.clang-format
    6. xcode选择 Edit -> Clang Format -> File

    NDK C++11 编译错误解决方法

    编译通过，但Eclipse还是提示错误

    其实NDk已经能很好的支持C++11了

    但是eclipse的代码分析器有时会分析不出来，仍然提示错误，

    这就会出现编译虽然通过了，但是还是运行不了，

    所以这时候要做的就是：把代码分析器干掉

    去到 Project > Properties > C/C++ General / Code Analysis

勾上Use project settings

把Coding Style、Syntax and Semantic Errors、Security Vulnerabilities 的勾去掉
